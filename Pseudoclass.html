Pseudo Classes

What are Pseudo-classes? A pseudo-class is used to define a special state of an element. For example, it can be used to: Style an element when a user mouses over it. Style visited and unvisited links differently. Style an element when it gets focus.

Text Properties

There are several text Properties. In this CSS we will use these.

Text-align
It is used to align/position the text. left, right, upper,lower,center etc
Text-transform
It is used to control the capitalization effects of the text
Text-indent
It is used to add space between the words
Text-decoration
It is used to apply decoration to the font used in the text elements
